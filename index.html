How this site works

One page, one API call.

No build step, no frameworks. The page you are reading is rendered from a single fetch to the public GitHub REST API.

fetch.js

// Pulls profile + repos straight from GitHub.
const user = await fetch(`https://api.github.com/users/SIRGPrice`)
  .then(r => r.json());

const repos = await fetch(`https://api.github.com/users/SIRGPrice/repos?per_page=100&sort=updated`)
  .then(r => r.json());

console.log(user.name, "·", repos.length, "repos");
